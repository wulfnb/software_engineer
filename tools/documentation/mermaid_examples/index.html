    Mermaid examples¶

    This document will show some common examples of diagrams created using Mermaid syntax.

    Flowchart¶

    ```mermaid
flowchart TD
    A[Start] --> B{Input Data}
    B --> C[Process Data]
    C --> D{Valid?}
    D -->|Yes| E[Save to DB]
    D -->|No| F[Log Error]
    E --> G[End]
    F --> G
    style A fill:#e1f5fe
    style G fill:#f1f8e9
```

    flowchart TD

    Sequence Diagram¶

    ```mermaid
sequenceDiagram
    participant U as User
    participant F as Frontend
    participant A as API
    participant D as Database

    U->>F: Submit Form
    F->>A: POST /api/data
    A->>D: INSERT query
    D-->>A: Success
    A-->>F: 201 Created
    F-->>U: Success Message
```

    Class Diagram¶

    ```mermaid
classDiagram
    class User {
        +String username
        +String email
        +Date createdAt
        +login()
        +logout()
    }

    class Post {
        +String title
        +String content
        +User author
        +publish()
        +addComment()
    }

    class Comment {
        +String text
        +User author
        +Date timestamp
    }

    User "1" --> "many" Post : creates
    User "1" --> "many" Comment : writes
    Post "1" --> "many" Comment : has
```

    ```mermaid

    FW --> WAF[WAF]

    LB --> GW[API Gateway]

    GW --> AUTH[Auth Service]
    AUTH --> JWT[JWT Validation]
    JWT --> POL[Policy Engine]
```
